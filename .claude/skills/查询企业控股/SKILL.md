# 查询企业控股信息

使用爬虫脚本查询 A 股汽车企业的股东持股比例数据。

## 数据来源

- **数据源**: AKShare 金融数据库（基于东方财富、新浪财经等公开数据）
- **公信力**: ⭐⭐⭐⭐ 高（来自公开披露的上市公司财报）
- **更新频率**: 季度更新（基于上市公司财报披露）

## 使用流程

1. **使用 AskUserQuestion 工具询问用户需求**：
   - 询问查询方式（全部车企 / 按公司名称 / 按股票代码）
   - 询问是否需要显示前10大股东详细信息
   - 询问是否需要保存到 CSV 文件

2. **执行查询**：
   根据用户选择，运行对应的命令：
   ```bash
   # 查询所有车企
   uv run .claude/skills/查询企业控股/crawler_stock_holdings.py

   # 按公司名称查询
   uv run .claude/skills/查询企业控股/crawler_stock_holdings.py --company 比亚迪

   # 按股票代码查询
   uv run .claude/skills/查询企业控股/crawler_stock_holdings.py --code 002594

   # 显示前10大股东详情
   uv run .claude/skills/查询企业控股/crawler_stock_holdings.py --code 002594 --detail
   ```

## 输出文件

数据将保存到 `out` 目录下的 CSV 文件，文件名格式：`企业持股比例_[日期].csv`

示例：`out/企业持股比例_20250131.csv`

包含以下字段：
- **公司名称**: 汽车企业名称
- **股票代码**: A 股代码
- **控股股东**: 第一大股东名称
- **持股比例**: 第一大股东持股比例
- **持股数量**: 第一大股东持股数量
- **实际控制人**: 企业实际控制人
- **性质**: 企业性质（国有控股/民营）
- **数据截至日期**: 数据更新日期

## 分析水军潜力

通过股东结构分析潜在水军：

1. **国有控股企业**（上汽、广汽、长安、江淮、海马）
   - 背后有政府支持
   - 可能存在组织化舆论引导
   - 但相对克制，注重品牌形象

2. **民营企业**（比亚迪、长城汽车）
   - 创始人控制（王传福、魏建军家族）
   - 市场化程度高
   - 营销投入大，粉丝文化活跃
   - **水军可能性较高**

3. **外资/港股背景**
   - HKSCC NOMINEES LIMITED 是香港中央结算公司
   - 代表海外投资者持股
   - 不直接参与舆论管理

## 注意事项

- 数据来自 AKShare API，需要联网获取
- 数据基于上市公司公开财报，有滞后性（季度更新）
- 部分国企的持股比例可能未披露（显示为"未披露"）
- 可以用 Excel 或 Python 读取 CSV 并做进一步的数据分析
- 水军分析仅供参考，需结合实际舆情数据
