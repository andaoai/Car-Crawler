# 查询汽车召回新闻

使用爬虫脚本查询国家市场监督管理总局（samr.gov.cn）的汽车召回新闻数据。

## 数据来源

- **官方机构**: 国家市场监督管理总局缺陷产品管理中心
- **网站**: https://qxzh.samr.gov.cn
- **公信力**: ⭐⭐⭐⭐⭐ 最高（政府官方数据源）

## 使用流程

1. **使用 AskUserQuestion 工具询问用户需求**：
   - 询问查询方式（全部召回 / 按关键词搜索）
   - 如果选择关键词搜索，询问关键词（品牌名、车型等）
   - 询问需要抓取多少页（默认5页，可选：3页、5页、10页、20页）
   - 询问是否需要按日期筛选（可选）

2. **执行查询**：
   根据用户选择，运行对应的命令：
   ```bash
   # 全部召回新闻
   uv run .claude/skills/查询召回新闻/crawler_samr_recall.py

   # 按关键词搜索
   uv run .claude/skills/查询召回新闻/crawler_samr_recall.py --keyword "关键词"

   # 抓取更多页数
   uv run .claude/skills/查询召回新闻/crawler_samr_recall.py --keyword "奔驰" --pages 10

   # 按日期筛选
   uv run .claude/skills/查询召回新闻/crawler_samr_recall.py --start-date "2024-01-01" --end-date "2024-12-31"
   ```

## 输出文件

数据将保存到 `out` 目录下的 CSV 文件，文件名格式：`召回新闻_[关键词]_[日期].csv`

示例：
- 全部召回新闻：`out/召回新闻_20250131.csv`
- 品牌筛选：`out/召回新闻_奔驰_20250131.csv`
- 车型筛选：`out/召回新闻_ModelY_20250131.csv`

包含以下字段：
- **新闻标题**: 召回新闻的标题
- **发布时间**: 召回公告发布的日期
- **涉及品牌**: 被召回的汽车品牌
- **一级总成**: 缺陷描述或涉及的部件
- **详情链接**: 召回新闻详情页的完整URL

## 命令行参数

| 参数 | 说明 | 示例 |
|------|------|------|
| `--keyword` | 搜索关键词（品牌名、车型等） | `--keyword "奔驰"` |
| `--pages` | 抓取页数（默认5） | `--pages 10` |
| `--page-size` | 每页数量（默认20） | `--page-size 50` |
| `--start-date` | 起始日期（YYYY-MM-DD） | `--start-date "2024-01-01"` |
| `--end-date` | 结束日期（YYYY-MM-DD） | `--end-date "2024-12-31"` |

## 常见关键词示例

- 品牌名：`奔驰`、`宝马`、`奥迪`、`大众`、`丰田`、`本田`、`特斯拉`、`比亚迪`、`蔚来`、`小鹏`
- 车型：`Model Y`、`卡罗拉`、`凯美瑞`、`帕萨特`
- 部件：`安全气囊`、`刹车`、`发动机`

## 注意事项

- 关键词搜索支持品牌名、车型等任意关键词
- 抓取完成后，告知用户数据已保存到的文件名
- 数据来源于政府官方渠道，权威性最高
- 可以用 Excel 或 Python 读取 CSV 并做进一步的数据分析
